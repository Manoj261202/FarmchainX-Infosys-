✨ Key Features

🔑 Authentication: JWT-based authentication with Role-Based Access Control (RBAC)
🛡️ Security: Spring Security integration with encrypted passwords
📊 Database: MySQL with Hibernate ORM (10 JPA repositories)
🎯 Traceability: Complete supply chain tracking from farm to consumer
🖼️ Media Management: Cloudinary integration for image storage
📱 QR Codes: Product verification via QR codes
🤖 AI Integration: Groq AI for product quality prediction
📚 Documentation: OpenAPI/Swagger UI for API exploration
